git init.					-> cria um repo git na pasta atual
git status					-> mostra o status do trabalho (de cada arquivo)
git add <nome_arquivo>		-> adiciona arquivo para ser comitado
git add .					-> adiciona todos os arquivos para ser comitado
git commit -m "msg"			-> faz commit do que estava no add com comentários
git log						-> mostra todo o histórico de commit
. 							-> é o diretório atual
git diff <nome_do_arquivo>	-> mostra a diferença entre o arquivo salvo e o arq modificado
git diff .					-> Mostra as diferenças de todos os arquivos no diretório
git reset <nome_do_arquivo>	-> Remove o arquivo do add
q							-> quit


## Fluxo de Trabalho

1. Criar o diretório (pasta) do projeto
2. Iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "mensagem"
8. volta para o passo "3"
